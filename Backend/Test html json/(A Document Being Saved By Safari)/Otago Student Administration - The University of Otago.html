My timetable

This page shows lectures, exams, and other academic events. The details of your timetable can change as you choose papers and as the University adjusts the year’s events to fit. Your timetable will be finalised at the start of semester, and room information will show at that point. You will receive an email when the timetable has been finalised. Any changes after that date (such as temporary room changes) will also be reflected on this page.
 

How to read the timetable

Each cell in the timetable shows:

  • the type of event (such as ‘Lecture’ — each type of event is colour-coded)
  • the paper code (such as ‘CHEM191’ for Chemistry paper 191)
  • the room code (such as ‘STDAV’ for St David Lecture Theatre — once the room has been decided) clicking this link will take you to a map of the location.

Hover over the cell to see the full paper and location information or click 'More..' when viewing on a phone.

If you have a clash, the two events will be shown alongside each other in the grid or with the same time on a phone.
 

How to change the timetable display

The default view of the timetable shows you all your events this week. Use the calendar and links above to move to the dates you want to see.

Use the Year and Month drop-down menus to change the current Year or Month. Select the week number in the calendar to view an entire week. Select an individual day in the calendar to view only that day. Select ‘Go’ to change the view.
